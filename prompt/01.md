1. Excluir Arquivos de Serviço e RotasEstes são os arquivos diretamente responsáveis pela integração com o GenieACS e pelas rotas de ONT.AçãoArquivoDescriçãoEXCLUIRservices/genieacs.jsContém toda a lógica de comunicação com a API do GenieACS.EXCLUIRcontrollers/ontController.jsContém a lógica de controle que interage com o GenieACS.EXCLUIRroutes/ont.jsDefine as rotas (endpoints) que acionam o ontController.Comandos de Exclusão (no diretório raiz do seu projeto):Bashrm services/genieacs.js
rm controllers/ontController.js
rm routes/ont.js
2. Limpar Dependências e Variáveis de AmbienteA. Modificar package.jsonVocê precisa remover qualquer dependência específica do GenieACS (provavelmente genieacs-api ou similar) que tenha sido adicionada para a comunicação.Abra o arquivo package.json.Remova a linha correspondente à dependência do GenieACS nas seções "dependencies" e/ou "devDependencies".Exemplo de linha a remover (pode variar):JSON{
  "dependencies": {
    // ...
    "genieacs-api": "^1.1.0", <--- REMOVER
    // ...
  }
}
B. Modificar .env.exampleRemova quaisquer variáveis de ambiente relacionadas ao GenieACS para garantir a limpeza do ambiente.Abra o arquivo .env.example.Remova as linhas que configuram o acesso à API do GenieACS.Linhas a remover (podem variar):# GenieACS Configuration
GENIEACS_HOST=http://localhost:7557
GENIEACS_USER=seu_usuario_api
GENIEACS_PASSWORD=sua_senha_api
3. Limpar Arquivos de InicializaçãoVocê precisa garantir que o servidor principal não tente carregar as rotas ou os arquivos que você acabou de excluir.A. Modificar server.js (ou index.js)Seu arquivo de inicialização do servidor (geralmente server.js ou index.js) precisa parar de importar o módulo de rotas da ONT.Abra o arquivo server.js (ou index.js).Remova a importação da rota: Procure e apague a linha que importa o roteador da ONT:JavaScriptconst ontRoutes = require('./routes/ont'); // <--- REMOVER ESTA LINHA
Remova o uso da rota: Procure e apague a linha que faz o aplicativo usar essas rotas:JavaScriptapp.use('/api/ont', ontRoutes); // <--- REMOVER ESTA LINHA
